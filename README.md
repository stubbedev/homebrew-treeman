# homebrew-treeman

Homebrew tap for [treeman](https://github.com/stubbedev/treeman) — a
per-worktree development environment helper.

## Install

```sh
brew install stubbedev/treeman/treeman
```

Or tap first, then install:

```sh
brew tap stubbedev/treeman
brew install treeman
```

## Updates

The `Formula/treeman.rb` file is regenerated automatically by the
[`release.yml`](https://github.com/stubbedev/treeman/blob/master/.github/workflows/release.yml)
workflow in the main repo each time a `v*.*.*` tag is pushed.
Do not edit it by hand — your change will be overwritten on the next
release.
